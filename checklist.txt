- [x] Migracja na model OKLCH: Precyzyjne sterowanie jasnością (L), nasyceniem (C) i odcieniem (H).
- [x] Odwrócona skala jasności: Konsekwentne stosowanie mapowania 0 = Biały (L: 1.0), 1000 = Czarny (L: 0.0).
- [x] Single Source of Truth: Wszystkie obliczenia i stan w engine.core.js.

2. LOGIKA GENEROWANIA KOLORÓW
- [x] Skale 0-1000: Wszystkie palety (Main, Additional, Functional, Badge) oparte na 11 krokach.
- [x] Granulacja Skali: Dynamiczne przełączanie widoku (co 10, 50, 100) bez utraty danych.
- [x] Tryb Absolute: Liniowy rozkład jasności.

- [x] Kolizje Odcieni: Automatyczna korekta barw funkcjonalnych względem koloru bazowego.
- [x] Ręczne Dodawanie Kolorów: Detekcja konfliktów Hue (<15°) z opcją automatycznej korekty.
- [x] Paleta Szarości: Szybkie dodawanie palety neutralnej (C=0).
- [x] Optymalizacja Dark Mode: Desaturacja kolorów (0.8x chroma) dla lepszej czytelności.

4. ZAAWANSOWANY WIDOK KONTRASTU
- [x] Dynamiczna Siatka Tła: 11 wierszy generowanych z pełnej skali (co-10) wybranej palety.

- [x] Gwarantowana Unikalność: Funkcja ensureUnique zapobiega nakładaniu się identycznych HEX w siatce.

- [x] Raportowanie Real-Time: Swatche pokazują wymagany kontrast (Min) oraz faktycznie uzyskany (Real).
